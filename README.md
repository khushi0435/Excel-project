# Excel Project: Student Study and Lifestyle Analysis

## 📌 Description
This project analyzes student lifestyle data collected using a Google Form and processed in Excel.

## 🎯 Objectives
- Analyze student study patterns
- Calculate performance using formulas
- Visualize data using Pivot Tables

## 🛠 Tools Used
- Microsoft Excel
- Google Forms
- GitHub

## 📊 Features
- Raw Data Sheet
- Calculated Fields (Max, Min, Average)
- Pass/Fail Logic
- Pivot Table Analysis

## 📁 Files Included
- Excel Sheet (Responses)
- Project Report (PDF)

## 📈 Result
The project helps understand student performance and lifestyle trends using Excel.
